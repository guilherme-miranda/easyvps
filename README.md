# EasyVPS

### Para Debian 8 64bits

- Nginx

- PHP 7.0

- MariaDB

- Let's Encrypt

- Git

- PhpMyAdmin/Adminer

- ISPConfig 3.1



## Instalando:
```
apt-get update && apt-get install git -y && git clone https://github.com/guilherme-miranda/easyvps.git && cd easyvps && chmod 755 install.sh.x && ./install.sh.x
```
